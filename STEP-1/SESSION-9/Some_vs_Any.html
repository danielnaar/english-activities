Using Some and Any

Learn the general rules for using these quantifiers correctly in positive, negative, and interrogative sentences.

General Rules

Use some in:

  • • Positive sentences
  • • Offers (Would you like...?)
  • • Requests (Can I have...?)
✓ I bought some apples at the market.
? Would you like some coffee?
? Can I borrow some money?

Use any in:

  • • Negative sentences
  • • Questions (Is there...? Do you have...?)
− I don't have any friends in this city.
? Are there any eggs in the fridge?
− We didn't get any sleep last night.

Quick Comparison

Structure Example with some Example with any
Positive Sentence I have some questions. Not applicable
Negative Sentence Not applicable I don't have any questions.
Question Would you like some water? (Offer) Do you have any water? (General question)

Common Mistakes

✕ I didn't get some presents.
→ ✓ I didn't get any presents.
✕ Do you have some milk?
→ ✓ Do you have any milk?
✕ She doesn't need some help.
→ ✓ She doesn't need any help.

Expressions

Compound Words

somebody / someone

I saw somebody in the hallway.

anybody / anyone

Is anyone home?

something / anything

Do you want something to eat?

somewhere / anywhere

Let's go somewhere fun this weekend!

Teacher Tip

You can create dialogues, roleplays or quick-fire games where students must choose between "some" and "any" on the spot. Ideal for oral practice!
