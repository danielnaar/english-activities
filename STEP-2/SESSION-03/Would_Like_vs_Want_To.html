Would like vs. want to

Both talk about desires. Would like is more polite and softer. Want to is stronger and casual.

Meaning & Intent

would like

Polite / Soft

Expresses a polite wish or an offer.

"I would like some tea."

VS

want to

Direct / Strong

Expresses a strong desire or a firm plan.

"I want some tea."

The Rules

Form 1: would like + noun

[ subject + would + like + noun ]

She would like a salad.

Form 2: would like + to + verb

[ subject + would + like + to + bare infinitive ]

They would like to order now.

Structure & Contractions

Type Structure Example
Affirmative She would like tea. She would like tea.
Negative She would not like tea. She would not like tea.
Question Would she like tea? Would she like tea?
Full Short Example
would 'd She'd like tea.
would not wouldn't They wouldn't like coffee.

Grammar Note: The main verb after would like to is always the bare infinitive: would like to go, would like to try.

Short Answers: Yes, she would. / No, she would not.

Would like vs. want to

Polite & Formal

Would like is polite in restaurants, shops, emails, offers and invitations.

Customer

"I would like to pay by card, please."

Direct & Everyday

Want to is direct and everyday. Use it with friends or when the tone is casual.

Friend

"I want to watch a movie tonight."

Offers and requests

Would you like...? Would you like to...? I would like...

"Would you like to sit here?"

"Yes, I'd love to. Thanks."

Patterns you will say a lot

  • • I would like some water, please.
  • • We would like to book a table.
  • • Would you like some tea?
  • • I want to go home now.
  • • He wouldn't like spicy food.

Common Mistakes

Do not conjugate would.

✕ Say she wills.
✓ Say she would.

After would like to use the bare infinitive.

✕ would like to going.
✓ would like to go.

Pro Tip: For things (nouns) you can drop to:
would like a pizza.

Mini Dialog

Roleplay

Server
Good evening. Would you like to see the menu?
Guest
Yes, please. I'd like a lemonade and we'd like to share a salad.
Server
Great. Anything else you want to try?
Guest
We want to try the soup later.
