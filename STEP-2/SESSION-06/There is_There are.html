There is / There are

Express existence clearly. Use these foundational structures to communicate that items, people, or concepts exist in a specific place or context.

Quick Rules & Structure

1
Affirmative

There is + singular / uncountable There are + plural noun
  • ✓There is a sofa.
  • ✓There is furniture. (Uncountable)
  • ✓There are two chairs.

2
Negative

There isn't + singular / uncount. There aren't + plural noun
  • ✓There isn't a rug.
  • ✓There aren't any shelves.

3
Questions

Is there...? Are there...?
  • Q:Is there a lamp?
  • A:Yes, there is. / No, there isn't.
  • Q:Are there any pictures?
  • A:Yes, there are. / No, there aren't.

Key Quantifiers

● SOME

Used in affirmatives

"There are some cushions."

● ANY

Used in questions & negatives

"Are there any curtains?" "There aren't any shelves."

● NO

Used to make a strong negative

"There is no mirror."

Model Sentences

Structure Model Sentence
Affirmative There is a coffee table in front of the sofa.
Affirmative There are three cushions on the sofa.
Negative There isn't a TV in the living room.
Negative There aren't any pictures on the wall.
Question Is there a mirror near the door?
Question Are there any chairs behind the table?
Short Answer Yes, there are. They are next to the window.
Short Answer No, there isn't. The lamp is on the shelf.
With Prepositions There is a rug under the table. There is a lamp beside the sofa.
